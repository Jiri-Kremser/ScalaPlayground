Scala Playground
================

some scala simple samples


This repo contains various Scala projects.

Usage
=====

For running the project use [sbt] (http://www.scala-sbt.org/release/docs/Getting-Started/Setup.html) 


**run main class:**

```
sbt run
```

**run interactive mode:**
```
sbt console
```

**run tests:**
```
sbt test
```
